Geelong Stars — Website Guide

A plain-English overview of how your website works

Where is the website hosted?

Your website is hosted on Cloudflare, one of the world's largest and most reliable internet platforms. Cloudflare serves your website from data centres all around the world, which means visitors get fast load times no matter where they are.

There are no traditional servers to maintain, and the hosting is included as part of Cloudflare's platform — so there are no separate hosting bills.

Your web address: www.geelongstars.com.au

The contact form

When someone fills out the contact form on the website, their message is sent directly as an email. This happens instantly and automatically — no third-party email service is involved.

Contact form emails currently go to: contact@geelongstars.com.au

Each email includes the person's name, email address, phone number (if provided), the subject they selected, and their message. You can reply directly to the sender from your inbox because the email is set up with a "reply-to" pointing back to whoever filled out the form.

The admin panel

Your website has a built-in admin area that allows authorised users to manage certain parts of the site without needing a developer. Through the admin panel you can:

  • Manage sponsors — add, remove, reorder, or update sponsor logos and details
  • Manage events — create, edit, or remove upcoming events
  • Upload images — add new photos to the site's image library
  • Edit page content — update text on various pages

You log into the admin panel with a username and password. If you need login details or a password reset, contact your website administrator.

Where is everything stored?

All your website's data is stored securely on Cloudflare's platform:

  • Text content, events, and sponsors are stored in a database managed by Cloudflare (called D1). This is like a digital filing cabinet for all the information displayed on the site.
  • Uploaded images (such as sponsor logos and event photos) are stored in Cloudflare's file storage (called R2). These are served quickly to visitors from Cloudflare's global network.
  • Static images (such as program photos and the club logo) are bundled directly with the website files and don't need a database.

How updates are made

There are two ways the website gets updated:

  • Content changes (sponsors, events, images, text) can be made by anyone with admin access through the admin panel. These changes appear on the website immediately.
  • Design or structural changes (new pages, layout changes, new features) require a developer. The website's source code is stored in a version-controlled repository, and changes are deployed to Cloudflare when ready.

Domain name

Your domain geelongstars.com.au is managed through Cloudflare's DNS. This means Cloudflare handles pointing your web address to the correct place. If the domain registration needs renewing, that is handled separately through the domain registrar (the company the domain was originally purchased from).

Good to know

  • The website is a single-page application, which means it feels fast and smooth — pages load without the full screen going blank between clicks.
  • The site is served over HTTPS (the padlock in your browser), so all traffic between visitors and the website is encrypted.
  • Cloudflare automatically provides protection against common internet attacks such as spam bots and denial-of-service attempts.
  • The site works well on mobile phones, tablets, and desktops.
